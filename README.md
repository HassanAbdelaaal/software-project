# HR System - Module 1

## Running the Application

### Backend
```bash
cd backend
npm run start:dev
```

### Frontend
```bash
cd frontend
npm run dev
```

## Git Commands

### Pull Latest Changes
```bash
git pull origin your-branch-name
```

### Push Your Changes
```bash
# Add your changes
git add .

# Commit with message
git commit -m "your message here"

# Push to GitHub
git push origin your-branch-name
```
